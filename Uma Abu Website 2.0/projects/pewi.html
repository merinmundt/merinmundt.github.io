PEWI Iowa State

Description

The People in Ecosystems/Watershed Integration (PEWI) tool is a simple web-based educational game designed to assist people in multidimensional land-use decision making. PEWI is an open-source project that can be utilized in classrooms and by land-managers or resource professionals to explore how different land cover types and configurations result in trade-offs in ecosystem services.

Motivation

I was hired to work on this project. It was my first software engineering project and so I was excited about creating an application that will be used to teach other students. I was also excited to see the impact of the project and how it can be applied in other aspects.

Lessons Learnt

I learned JavaScript for this project and it was my first time working with Github. I learned about working in an Agile Scrum manner and learned about one of the different ways computer science can be applied in various fields.

The application can be found at PEWI. For more information, visit PEWI Iowa State.
